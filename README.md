# E2E_Agentic_AI_QA_Workflow

## 🎯 Summary Of This Project
In this repository I'll perform:
- Read User Stories automatically
- Generate AI-powered Test Plans
- Perform Exploratory Testing using Playwright MCP
- Auto-generate Playwright Automation Scripts
- Execute tests and auto-heal failures
- Generate complete test reports
- Commit all artifacts to GitHub automatically

## 🧠 AI Agents Used in This Workflow
This Agentic QA pipeline uses the following agents:
• Test Planner Agent – Generates test scenarios from requirements
• Test Generator Agent – Converts test plans into Playwright scripts
• Test Healer Agent – Fixes failing tests automatically

## 🧰 Tools & Technologies Used
• Playwright Test Automation
• Playwright MCP Server
• GitHub MCP Server
• AI Agents for QA Automation
• Node.js
• JavaScript Test Scripts
